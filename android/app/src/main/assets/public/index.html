About

Lava Prompt Lab is a mobile-first prompt synthesis console, designed to be wrapped as a Capacitor Android APK for on-device use on Galaxy S24+ class hardware.

HTML/CSS/JS only, no backend required. All state is stored locally, ready to be embedded in a WebView for native distribution.
